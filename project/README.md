# Предсказание задержек авиарейсов (Flight Delay Predictor)

**Автор:** Карьева Мария  
**Группа:** `<УИБО-02-23>`  
**Контакт:** `<marikarjeva17@gmail.com>`

Краткое описание:

Проект посвящён построению сервиса прогнозирования задержек авиарейсов. По характеристикам рейса (время вылета, день недели, аэропорт, авиакомпания, расстояние) модель предсказывает задержку в минутах. Используются синтетические данные, две модели (`LinearRegression` и `CatBoost`), FastAPI-сервис, Docker, тесты и логирование.

Результат — REST API, который по параметрам рейса возвращает прогноз задержки и категорию.

# 4. Как запустить проект

## 4.1. Запуск обучения модели

```bash
cd project
python src/eda_and_train.py
```

Скрипт генерирует синтетические данные (50 000 записей), проводит EDA, обучает две модели (`LinearRegression` и `CatBoost`), сравнивает метрики (**MAE**, **R²**, точность ±15 минут) и сохраняет лучшую модель в папку `artifacts/`.

## 4.2. Запуск сервиса (API)

### Локально

```bash
cd project
uvicorn src.app.api:app --reload
```

Сервис поднимается на порту `8000`.

### Эндпоинты

- `GET /health` — проверка состояния сервиса и статуса загрузки модели.
- `POST /predict` — предсказание задержки по параметрам рейса.

### Проверка работоспособности

Откройте в браузере:

```text
http://127.0.0.1:8000/docs
```

### Через Docker

Сборка образа:

```bash
cd project
docker build -t flight-delay-api .
```

Запуск контейнера:

```bash
docker run -p 8000:8000 flight-delay-api
```

# 5. Данные

## Источник данных

Синтетический датасет (50 000 записей), моделирующий работу крупных аэропортов:

- JFK
- LAX
- ORD
- ATL

и авиакомпаний:

- Delta
- American Airlines
- Southwest Airlines

Расположение:

```text
data/flights.csv
```

Файл генерируется автоматически при первом запуске `src/eda_and_train.py`.

## Используемые признаки

- час вылета (циклический sin/cos);
- день недели;
- месяц;
- флаг выходного дня;
- расстояние;
- лаг-задержка аэропорта;
- загруженность аэропорта;
- средняя задержка авиакомпании.

## Целевая переменная

`DEPARTURE_DELAY` — задержка вылета в минутах.

Значение ограничивается диапазоном:

```text
[-60, 180]
```

## Замена на реальный датасет

Для использования реального датасета (например, US DOT Flight Delays с Kaggle) достаточно поместить CSV-файл с аналогичной структурой колонок в папку `data/`.

   - `CatBoost`

```bash

```
